# Naz Lab Global Language Policy

## Priority

Naz Lab is Bangla-first by default.

1. Bangla first
2. English second
3. Other languages optional

## Bangla requirements

Bangla output must be natural, spoken, netizen-friendly, Facebook-ready, and voiceover-ready.

Default Bangla style should be simple, human, and usable, not stiff textbook Bangla.

When regional flavor is useful, the primary default is Rangpur / Nilphamari / North Bengal tone.

Secondary regional tones may be supported when requested:

- Dhakaiya
- Chattogram
- Sylhet
- Noakhali / Comilla

## English requirements

English should be clean, practical, and ready to use.

True Noir Tales and ToolFlow are English content projects by default, but the overall Naz Lab system remains Bangla-first.

## Workstation behavior

General workflows should default to Bangla.

English should be used when:

- the user requests English
- the selected project preset is True Noir Tales
- the selected project preset is ToolFlow
- the content target is clearly English-first

## Image policy language note

Image prompts should support Bangla-first planning and English generator-ready prompts when useful.

## Voice and video policy language note

Voice and video workstations should default to Bangla for general content, while preserving English presets for True Noir Tales and ToolFlow.
