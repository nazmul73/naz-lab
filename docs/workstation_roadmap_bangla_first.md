# Naz Lab Workstation Roadmap - Bangla First

Naz Lab default language priority:

1. Bangla first
2. English second
3. Other languages optional

General workflows should default to Bangla.

English is still supported for selected project presets and user requests.

Current stable phases:

- Phase 0 Foundation
- Phase 1 Text Workstation
- Phase 2 Master Dashboard
- Phase 3 Image Workstation
- Phase 4 Voice Workstation

Next phase:

- Phase 5 Video Workstation

Video Workstation should include:

- project preset selector
- content type selector
- language selector
- Bangla-first default
- image package placeholders
- voice package placeholders
- scene sequence builder
- shot list builder
- subtitle style builder
- edit style builder
- copy-ready video package
- future video backend placeholder

Bangla requirements:

- natural spoken Bangla
- Facebook-ready Bangla
- netizen-friendly Bangla
- voiceover-ready Bangla
- Rangpur / Nilphamari / North Bengal flavor when regional tone is useful
