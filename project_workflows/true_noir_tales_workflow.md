# True Noir Tales End-to-End Workflow

True Noir Tales is a true-crime / noir storytelling project. It can remain English-first as a project preset, while Naz Lab as a whole remains Bangla-first.

## Global language behavior

- Naz Lab default: Bangla-first
- True Noir Tales preset: English-first by default
- Bangla version: supported whenever requested
- Other languages: optional

## Safety and content rules

Use adult-focused storytelling only.

Do not generate or request:

- gore
- blood-focused visuals
- dead bodies
- visible wounds
- exposed violence
- child-victim framing
- sensational violence

For visual prompts:

- adult subjects only
- Bangladeshi people and settings when the story is local/general
- no police logo, watermark, or fake official marks
- no sindoor unless explicitly requested
- realistic cinematic noir style
- emotional tension without graphic content

## Workflow chain

1. Story/topic intake
2. Script package
3. Image prompt package
4. Voice package
5. Video package
6. Caption/CTA/hashtags
7. Final posting package

## Step 1 - Story/topic intake

Input can be:

- short topic
- true-crime summary
- social post idea
- narration draft
- case-style story

Required fields:

- topic
- language target
- platform
- length
- content safety notes

## Step 2 - Script package

Recommended structure:

- Hook
- Context
- Tension
- Turn/reveal
- Psychology line
- Question CTA

English default style:

- natural American Grade 7 English
- suspenseful but restrained
- documentary/noir tone
- no gore description
- no sensational wording

Bangla version style:

- natural spoken Bangla
- Facebook-ready
- netizen-friendly
- voiceover-ready
- simple, emotional, factual

## Step 3 - Image prompt package

Prompt direction:

- realistic cinematic photography
- true crime noir atmosphere
- dramatic shadows
- emotionally tense adult human-centered scene
- Bangladesh setting by default when relevant
- no gore, no blood, no dead body, no visible wounds
- no sindoor unless explicitly requested

Recommended output formats:

- Reel thumbnail: 9:16
- Facebook post: 1:1 or 4:5
- Carousel cover: 1:1
- Story scene: 9:16

## Step 4 - Voice package

Voice direction:

- suspenseful but controlled
- no overacting
- documentary storytelling cadence
- short dramatic pauses
- clear pronunciation

Voice mode:

- generic/original voice direction by default
- reference voice profile only if user provides or explicitly authorizes reference audio

## Step 5 - Video package

Video structure:

- Scene 1: hook visual
- Scene 2: context visual
- Scene 3: critical detail
- Scene 4: tension/reveal
- Scene 5: question CTA

Edit style:

- cinematic suspense
- documentary reel
- subtle zooms
- readable subtitles
- restrained music/sound direction

## Step 6 - Caption/CTA/hashtags

Caption style:

- short, curiosity-driven
- ends with a question when useful
- no clickbait that misrepresents facts

CTA examples:

- What would you have noticed first?
- Do you think this was a warning sign?
- Which detail changed the story for you?

Hashtag examples:

- #TrueCrime
- #CrimeStory
- #Mystery
- #NoirStory
- #CrimePsychology
- #DocumentaryStyle

## Final posting package fields

- project_preset
- language
- topic
- platform
- script
- image_prompt
- voice_direction
- video_direction
- caption
- CTA
- hashtags
- safety_notes
- output_paths
