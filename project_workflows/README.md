# Naz Lab Project Workflows

This folder contains end-to-end workflow documentation and JSON templates for specific content projects.

## Current workflows

### True Noir Tales

Files:

- `true_noir_tales_workflow.md`
- `true_noir_tales_package_template.json`

Purpose:

```text
Story/topic
-> script package
-> image prompt package
-> voice package
-> video package
-> caption/CTA/hashtags
-> posting package
```

Default language:

- English-first project preset
- Bangla supported when requested

Core safety:

- adult-focused storytelling
- no gore
- no dead body
- no visible wounds
- no exposed violence
- no sensational violence

### ToolFlow

Files:

- `toolflow_workflow.md`
- `toolflow_package_template.json`

Purpose:

```text
Tool/topic
-> content angle
-> main post
-> carousel
-> reel script
-> image/cover prompt
-> voice package
-> video package
-> caption/CTA/hashtags
-> posting package
```

Default language:

- English-first project preset
- Bangla supported when requested

Brand voice:

- modern
- clean
- practical
- premium
- trustworthy
- useful
- non-hype

Avoid:

- fake income claims
- spammy marketer tone
- unverified AI news
- crypto speculation
- generic motivation fluff

## Global Naz Lab rule

Naz Lab itself is Bangla-first by default, even when some project presets are English-first.
